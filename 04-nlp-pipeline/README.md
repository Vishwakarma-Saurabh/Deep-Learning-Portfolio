# 🧠 Project 4: Sentiment Analysis + Text Generation

> NLP pipeline using LSTMs for sentiment classification and text generation

## 🎯 Overview

This project implements a complete NLP pipeline with:
1. **Sentiment Analysis** - Classify movie reviews as Positive/Negative
2. **Text Generation** - Generate new text from seed phrases

## 🏗️ Architecture
Text → Tokenize → Embedding → LSTM → Dense → Output


## 🚀 Quick Start
pip install -r requirements.txt
python main.py


🔑 Key Concepts
LSTM/GRU - Sequential data processing
Word Embeddings - Word → Vector representations
Tokenization - Text → Tokens
Sentiment Analysis - Text classification
Text Generation - Autoregressive generation
